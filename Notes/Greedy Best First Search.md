# Greedy Search
Expands the node that appears to be closest to the goal based on the evaluation function h(n) i.e. expands the lowest h(n) values first.

Completeness: No
Optimality: No
Time Complexity: $O(b^m)$
Space Complexity: $O(b^m)$
